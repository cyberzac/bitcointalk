Bitcoin is money

  • Medium of exchange: traded for other stuff
  • Unit of account: fungible, all bitcoins are the same
  • Unit of account: divisible down to 100 Millionth, a 'Satoshi'
  • Store of value: Bitcoins are scarce, currently ~12 million, max 21 million
  • Market value of all currently existing Bitcoins is ~2 Billion USD

Bitcoin is money on the net

Transactions are

  • Fast, a few seconds
  • Cheap, currently about ~10 öre
  • Unstopable
  • Non-reversible

Bitcoin is a protocol

  • Open source
  • Peer to peer
  • No single point of failure

Where is it used

  • Dice rolling, with guaranteed honesty
  • Rewards for work
  • Anonymous donations (Anonymous, Occupy movement)
  • Unstoppable donations (WikiLeaks, PirateBay)

More usage

  • Betting
  • Pay for online dating account
  • Buy network assets (domain names, cloud storage, CPU, hosting)
  • Buy just about anything

How are they created?

  • 25 new Bitcoins are created on average each 10 minutes, in a process called "mining"
  • Mining is the way new Bitcoins are distributed
  • Mining validates transactions
  • Anyone can mine!

How do you get Bitcoins?

  • Mine them
  • Earn them
  • Buy and sell them, most often at MtGox or BitStamp
  • In Sweden at Btcx, Kapiton and Safello


Bitcoins are quite hard to get, but really easy to use

Bitcoin Addresses

Example: 13dhfDmPEvMXCBs44rU6vCuTDK42rVoTtw

  • Is the 160-bit hash of the public part of a key pair
  • Contains checksums, 2^-32 risk of misstyped address
  • Can be created offline
  • Cannot be directly traced back to you

Transactions

  • A collection of inputs and outputs
  • The private key is needed to sign a transaction using the public address as input
  • The inputs are outputs from a previous transaction
  • An output contains a value and a script, which can be just an address, or a program!
  • Inputs = outputs + fee

Solving double spending

  • Timestamp
  • Distributed database
  • This is mining

Blocks - Mining output

  • Contains a header + a list of transactions
  • The SHA256 hash of the blockheader must be below the current target
  • The miners tries to generate a correct blockheader hash, by trial and error
  • The miners are currently rewarded 25 BTC + fees per block

Difficulty

  • Current Network Hashrate: 2.6 PHash/s
  • Adjustment every 2016 blocks, about 2 weeks
  • Adjusted to achieve 6 block per hour
  • Hashing capacity grows ~4% every day (mid October 2013)

Target

  • difficulty = maximum_target / current_target
  • maximum_target is 2^256
  • Current target
  • Current difficulty
  • Probability for one attempt
